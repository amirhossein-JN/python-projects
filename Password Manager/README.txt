# ATM Simulator

A console-based ATM Simulator developed in C++ using Object-Oriented Programming (OOP). The application allows users to log in using a PIN, check their account balance, deposit and withdraw money, and change their PIN through an interactive menu.

## Features

- Secure PIN login
- Check account balance
- Deposit money
- Withdraw money
- Change PIN
- Interactive menu
- Input validation

## Technologies

- C++
- Object-Oriented Programming (OOP)

## Concepts Used

- Classes & Objects
- Constructors
- Encapsulation
- Member Functions
- Loops
- Switch Statements
- Conditional Statements
- User Input / Output
